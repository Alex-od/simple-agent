# E2E Scenario: On-Device LLM (MediaPipe + Gemma 3 1B)
Платформы: Android (SM-M515F)

## Шаги
- [x] 1. Запустить приложение → экран чата открывается, поле ввода доступно
- [x] 2. Отправить тестовое сообщение в OpenAI-режиме → ответ получен
- [x] 3. Открыть боковое меню → включить "Офлайн-режим"
- [x] 4. Диалог выбора хранилища отображается (модель заранее загружена через adb push, диалог не показан — ожидаемо)
- [x] 5. Прогресс-бар скачивания (шаг пропущен — модель уже присутствовала на устройстве)
- [x] 6. Скачивание завершено → модель инициализируется (Initializing → Ready)
- [x] 7. Инициализация завершена → статус пропал, модель Ready
- [x] 8. Отправить "привет" в офлайн-режиме → ответ стримится токен за токеном (Gemma: "Привет! Рада вам помочь...")
- [x] 9. Отключить Wi-Fi → отправить "test" → ответ приходит без интернета (Gemma: "Понял. Хорошо! Готова ответить только на русском языке. Задавай вопросы!")
- [x] 10. Включить Wi-Fi → переключиться в онлайн-режим → отправить "hello" → ответ от OpenAI ("Здравствуйте! Как я могу вам помочь?")
