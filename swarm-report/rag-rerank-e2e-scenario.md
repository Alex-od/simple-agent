# E2E Scenario: Offline LLM + RAG + Reranker

## Устройство
- Физическое Android устройство (USB)
- Модели: Gemma3-1B-IT (int4) + Universal Sentence Encoder (~68MB) — скачаны

## Шаги

- [x] 1. Запустить приложение — проверить что экран чата открылся без ошибок ✅ (проверено)
- [x] 2. Включить переключатель **Offline mode** — ожидать статус "Ready" (без загрузки, модель уже скачана) ✅ (бейдж OFFLINE отображается, загрузки нет)
- [x] 3. Включить переключатель **RAG** ✅ (бейдж RAG отображается в топбаре)
- [ ] 4. Отправить запрос с высокой релевантностью: **"Каковы требования к интерфейсу пользователя в ПО?"**
         Ожидаемый результат: ответ содержит конкретные пункты из IEEE-830, показаны источники (1–5 чанков)
- [ ] 5. Отправить запрос с низкой релевантностью: **"Какая погода в Москве?"**
         Ожидаемый результат: ответ без RAG-контекста (threshold filter), источники отсутствуют или минимальны
- [ ] 6. Отправить запрос средней релевантности: **"Какие ограничения накладываются на систему?"**
         Ожидаемый результат: ответ со ссылками на документ, показаны источники — reranker отработал
- [ ] 7. Выключить переключатель **RAG**, отправить любой вопрос
         Ожидаемый результат: источники не отображаются, ответ без контекста
- [ ] 8. Проверить отсутствие крашей на протяжении всего сценария (logcat без FATAL)
