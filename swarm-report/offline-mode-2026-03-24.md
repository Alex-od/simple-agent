# Отчёт: Офлайн-режим (Ollama)
Дата: 2026-03-24

## Описание задачи
Реализация офлайн-режима в приложении SimpleAgent — возможность переключаться между OpenAI и локальной моделью Ollama (llama3.2:3b), а также комбинировать с RAG-поиском по документам.

## Что реализовано

- `OllamaService.kt` — сервис для обращения к Ollama API (`/v1/chat/completions`)
- `ChatService.kt` — общий интерфейс для ChatService
- `ChatRepositoryImpl.kt` — поддержка переключения между OpenAI и Ollama
- `AppModule.kt` — DI для обоих сервисов
- `SendMessageUseCase.kt` — передача флага офлайн-режима
- `ExtractTaskStateUseCase.kt` — обновлён под новый стек
- `ChatScreen.kt` / `ChatView.kt` / `ChatViewModel.kt` — UI переключателей и бейджей
