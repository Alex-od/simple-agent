# E2E Scenario: RAG — источники и цитаты
Платформы: Android

## Шаги

- [x] 1. Запустить сборку проекта — убедиться что компилируется без ошибок ✅ (проверено, BUILD SUCCESSFUL, юнит-тесты прошли)
- [ ] 2. Открыть приложение на устройстве/эмуляторе
- [ ] 3. Открыть меню (кнопка hamburger) — убедиться что переключатель RAG присутствует
- [ ] 4. Включить переключатель "Поиск по документам" (RAG) — в заголовке должен появиться бейдж RAG
- [ ] 5. Отправить тестовый вопрос (по теме документов в базе знаний)
- [ ] 6. Дождаться ответа ассистента — проверить что ответ появился
- [ ] 7. Проверить что под пузырём ассистента видна кнопка "Источники (N)" — N > 0
- [ ] 8. Нажать кнопку "Источники (N)" — список должен раскрыться
- [ ] 9. Проверить что каждый источник отображает: имя файла + номер чанка + цитату в кавычках «...»
- [ ] 10. Повторно нажать "Источники (N)" — список должен свернуться
- [ ] 11. Выключить RAG, отправить сообщение — блок "Источники" не должен появляться
